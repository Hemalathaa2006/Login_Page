# Login Page

A static login page with client-side validation built using HTML, CSS, and JavaScript.

## Features
- Gmail-only email validation (example@gmail.com)
- Password validation (minimum 6 letters or numbers)
- Clean and modern card-based UI
- Displays real-time success and error messages
- Responsive and user-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript

## Validation Rules
- Email must contain `@gmail.com`
- Password must be at least 6 characters (letters or numbers)

## How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
